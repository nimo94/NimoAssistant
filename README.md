# NimoAssistant  
**Nimo — The Educational A.I. Virtual Assistant**  

Welcome to **NimoAssistant**, your personal educational A.I. that helps you learn, explore, and have fun!  
Enjoy exciting updates and intelligent features designed just for you.  

---

## Features & Commands

You can interact with Nimo using simple commands:

| Command | Description |
|----------|-------------|
| `*shut down` | Shuts down the assistant. |
| `*close` | Closes **NimoAssistant.exe**. |
| `*Nimo` | cleans cpmmand shell. |
| `*mode` | Switch between normal mode and **learning mode** (you can teach Nimo new things). |
| `*google` | Opens Google in your browser. |
| `*logout` | Signs you out of the assistant. |
| `*reset` | Resets your saved data (name, age, gender, and search history). |

---

## Learning Mode

In **Learning Mode**, you can teach Nimo new commands or responses.  
Just type `mode` to toggle between **Normal Mode** and **Learning Mode**.
